## 结构体
一般用的变量互相独立，无内在联系，但有些数据有内在联系，成组出现，例如一个学生的学号，姓名，性别……  
这种一般无法用数组存储，因为数组一个只能存放同一类型的数据，所以用户可以建立由不同类型数据组成的组合型的数据结构，它称为结构体。  

声明一个结构体类型的一般形式为：  
```
struct 结构体名
{
	成员1;
	成员2；
	……
	成员n;
};
```
例如：图书的标题、作者、价格、日期、
```
struct Book
{
	char title[128];
	char author[40];
	float price;
	unsigned int date;
	char publisher[40];
};
```
定义结构体变量时，可以在末尾;前定义变量名，也可以在main函数中定义
区别是一个是全局变量，另一个则是局部变量。 
成员声明具体如下：   
```
类型名 成员名;
```

## 访问结构体
访问结构体成员需要用到.点运算符具体用法如下：  
```
结构体名.成员名
```
注意：每一个成员是结构体中的一个域。
